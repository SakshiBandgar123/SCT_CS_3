# 🔐 Password Strength Analyzer

A simple and interactive Password Strength Analyzer built using **HTML, CSS, and JavaScript**. This project evaluates the strength of a password based on common security criteria and provides users with feedback to create stronger passwords.

## 📌 Features

- Checks password strength in real time
- Evaluates password length
- Detects uppercase and lowercase letters
- Detects numbers and special characters
- Provides feedback for improving password security
- Simple and user-friendly interface

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any modern web browser.
4. Enter a password to analyze its strength.
